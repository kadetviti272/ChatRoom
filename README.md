# Java Chat Client
## Клиент-серверный чат
### v.0.1

Состоит из двух блоков:

* Отдельные онка Регистрации\Авторизации\Чата
* Шифрование паролей
* Общая чат-сессия с возможностью писать личные сообщения
* Дизайн сообщений в виде облака
